### Netflix Data Cleaning & Visualization Project

This project focuses on **cleaning and analyzing a real-world Netflix dataset** with an emphasis on **handling missing data without removing any rows or columns**.

The original dataset contained missing values across multiple columns, including:

 - director, cast, country
 - date_added, rating, duration

Using **Python and Pandas**, all missing values were handled systematically while preserving **100% of the original data**.

**Result after cleaning:**

 - All columns contain **0 null values**
 - No rows or columns were dropped
 - Dataset integrity fully preserved

This approach improves data reliability and avoids bias caused by data deletion.
